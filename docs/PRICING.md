## 三档方案

### 🔹 Starter（个人 / 试用）
- **价格**: 免费
- **限制**: 200 文档 / 月
- **包含**: 基础 RAG 检索 + 图谱可视化
- **支持**: 社区支持
- **适用**: 个人研究者、小规模试用

### 🔸 Pro（团队 / 中小企业）
- **价格**: ¥3,000 / 月
- **包含**:
  - 10,000 文档 / 月
  - Web Demo (Streamlit Cloud 或本地)
  - LangChain BaseRetriever 集成
  - 双数据集 (zh + CAIL 真实案件)
  - 邮件支持 + 4 小时响应 SLA
- **适用**: 法律科技初创、文档分析团队

### 🔶 Enterprise（大型企业 / 私有部署）
- **价格**: 联系销售
- **包含**:
  - 无限文档
  - 私有部署 (Docker / K8s)
  - 自定义实体抽取 + 链接模型
  - 专属客户经理 + 1 小时响应 SLA
  - 模型微调与定制
  - 数据驻留本地
- **适用**: 律所、检察院、大型科技公司
