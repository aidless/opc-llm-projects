## 4. 关键洞察

### 4.1 CAIL vs zh 对比

1. **CAIL 检索精度更优** (100% vs 62.5%)：
   - 真实案件事件类型清晰，query 子串匹配高
2. **CAIL char_f1 显著更高** (0.66 vs 0.01)：
   - 真实案件 query 与 text 字符级重叠度更高
3. **CAIL 有真实参与方** (200 vs 0)：
   - 真实法官人名是 zh 合成数据缺失的关键维度
4. **zh sub-category top-k = 0**：
   - 合成数据 query 设计过于通用，sub-category 检索无意义

### 4.2 工程技巧

- **法官去重**：用 200+ 中国复姓池（宇文/长孙/司马/上官等）替换重复名
- **v2 评测模块化**：每个指标独立函数 + main 编排
- **年份+split 维度**：便于训练/测试集质量评估
